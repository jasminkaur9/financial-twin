# Financial Twin — DRIVER Project

Multi-model AI financial simulation: GPT-4o, Gemini Pro, Claude each model your financial future
with different assumptions. See where they agree, where they diverge, and why.

## Workflow (DRIVER)

1. `/finance-driver:define` — Vision + 分头研究 ✅
2. `/finance-driver:represent-roadmap` — Break into buildable sections
3. `/finance-driver:implement-screen` — Build and run, iterate
4. `/finance-driver:validate` — Cross-check instruments
5. `/finance-driver:evolve` — Final export package
6. `/finance-driver:reflect` — Capture lessons

## Philosophy

**Cognition Mate (认知伙伴):** 互帮互助，因缘合和，互相成就

Neither creates alone. Meaning emerges from interaction.

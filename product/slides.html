Key Technical Decisions
  • instructor — all 3 models return identical Pydantic schema. Without it, can't compare.
  • ThreadPoolExecutor — 4 seconds total, not 18 sequential
  • google-genai not google-generativeai — deprecated, caught in research
  • Demo mode — same numpy-financial engine, no API keys needed
CI/CD
  • GitHub Actions — lint + 21 tests on every push
  • pytest — known-answer validation for every formula
  • Secrets scan — fails build if API key accidentally exposed
